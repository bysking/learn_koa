apicloud vue 项目结构

├─build //webpack编译配置
│ ├─const.js // webpack编译常量
│ ├─apis // api 环境地址
│ ├─generate-file-webpack-plugin // webpack插件（根据配置生成文件）
│ ├─utils // 脚手架工具方法
│ ├─webpack.base.config.js // webpack基础文件
│ ├─webpack.dev.config.js // webpack开发环境配置
│ ├─webpack.prod.config.js // webpack生产环境配置
├─config // apicloud 配置文件及首页模板文件
├─src // 项目源文件总目录
│ ├─assets // 通用样式目录
│ ├─components // 通用组件目录
│ ├─router // 路由
│ ├─vendors // 公共依赖目录
│ ├─view // 源文件目录
├─static // 静态资源，记得图片需要require进来
├─widget 打包目录

index.html中

script里面：apiready = function () {} 函数体内进行调用以下的函数

parseTapmode
解析元素 tapmode 属性，优化点击事件处理

默认页面加载完成后，引擎会对 dom 里面的元素进行 tapmode 属性解析，若是之后用代码创建的 dom 元素，则需要调用该方法后 tapmode 属性才会生效

parseTapmode()

api.parseTapmode();

// 设置状态栏样式
api.setStatusBarStyle

// 保持屏幕长亮
api.setKeepScreenOn

ejs: https://ejs.bootcss.com/

config // apicloud 配置文件及首页模板文件

安装： 

npm install ejs

使用

var ejs = require('ejs'), // 引入
people = ['geddy', 'neil', 'alex'], // 定义数据
html = ejs.render('<%= people.join(", "); %>', {people: people}); 渲染模板

条件：凡是使用的js代码的地方，都要用<%.....%>包裹，取出值的时候：<%=.......%>
<% if (user) { %>
<h2><%= user.name %></h2>
<% } %>
